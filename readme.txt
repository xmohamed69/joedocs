JoeLink AI – Desktop Client for JoeDocs
======================================

Version: 1.0.0
Date: [Date]

1. Overview
-----------

JoeLink AI is the official Windows desktop client for the JoeDocs document management platform.

It provides:
- A convenient desktop window for accessing your JoeDocs organization,
- Quick access to the document explorer and user management,
- AI-assisted features such as smart document naming, organization, and summaries.

This desktop client communicates with your organization’s JoeDocs backend over the internet. It does NOT store your documents locally, except for temporary caching handled by the operating system or browser engine.

You need a valid JoeDocs organization and user account to log in and use this client.

2. System Requirements
----------------------

- Operating System: Windows 10 or later (64-bit recommended).
- Internet Connection: Required to access the JoeDocs backend.
- CPU & RAM: Sufficient for typical office/workstation use.
- Disk Space: Enough to install the client (~50–200 MB), plus any temporary browser cache.

3. Installation
---------------

1. Download the installer:
   - From our official website, download `JoeLinkAISetup.exe` (or the latest available version).

2. Run the installer:
   - Double-click `JoeLinkAISetup.exe`.
   - If Windows asks for confirmation (SmartScreen), verify the publisher is [Your Company Name] and click "Run anyway" if you trust the source.

3. Follow the setup wizard:
   - Choose your preferred language (English or French, if available).
   - Click "Next" to proceed through the steps.
   - Choose the installation directory (default: `C:\Program Files\JoeLinkAI\`).
   - Optionally select "Create desktop icon".
   - Click "Install" to begin the installation.
   - Once finished, click "Finish" to close the wizard, and optionally launch JoeLink AI.

4. First Launch & Login
-----------------------

1. Start JoeLink AI:
   - From the Start Menu: Start → JoeLink AI → JoeLink AI.
   - Or from the Desktop if you created a shortcut.

2. Login screen:
   - You will see the JoeDocs login page inside the application window.
   - Enter:
     - Your Organization ID (10 digits),
     - Your User ID (10 digits),
     - Your password.

3. Organization setup:
   - If you do not yet have an organization, click the **“Add organisation”** or similar link.
   - This will open the official JoeDocs website in your default browser, where you can:
     - Choose a plan,
     - Enter your organization details,
     - Provide Baridi Mob payment information,
     - Submit a request to our team.
   - After verification, you will receive your organization ID and initial super user credentials by email.

5. Using JoeLink AI
-------------------

Once logged in:

- The **left sidebar** typically shows:
  - Your profile (name, profession tag, user ID, organization name & ID),
  - Navigation links (Documents, Users, Dashboard, Logs for owners).

- The **main area** shows the **document explorer**:
  - Folder groups (e.g. HR, Finance, Legal),
  - Folders and vaults,
  - List of documents.

Common actions:
- Upload documents (OWNER/EDITOR only),
- View documents with inline previews (PDF, images, text, DOCX text preview),
- AI assistant:
  - Suggests better document titles based on content,
  - Helps auto-organize docs into appropriate folders,
  - Generates professional “Executive Summary” text for documents.

Your permissions (view, print, move, delete) depend on:
- Your role (OWNER/ORG_ADMIN/EDITOR/VIEWER),
- Per-document flags (whether the document is allowed to be printed/moved/deleted),
- Access control lists (ACLs) set by your organization’s owner/admin.

6. Uninstallation
-----------------

To uninstall JoeLink AI:

1. Open Windows "Apps & features" (or "Programs and Features" on older versions).
2. Find "JoeLink AI" in the list.
3. Click "Uninstall" and follow the prompts.

This will remove the desktop client but not your documents or organization data from the JoeDocs server.
